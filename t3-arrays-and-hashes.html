    Arrays and Hashes

    Arrays and hashes are both collections of objects. Arrays are simply a ordered list of objects. They are created by listing objects seperated by commas inside square brackets. For example an array of numbers:

    numbers = [1, 2, 3]

    Anything can be made into an array:

    animals = [duck, cat, moose]

    Arrays are integer indexed starting at 0. animals[1] would return the second animal on our list cat. animals[-1] would return moose as ther list loops into itself.

    Hashes are also a way to organize objects. They do it a little differently though. Hashes are made up of key-value pairs. The indexing is done with created keys instead of integers. Hashes are made using curly brackets:

    animals = {type => cat}

    The keys and values can be anything you want.

    animals = {type: "cat", color: "black", fur => "long" }

    Values in hashes can be called by refrencing the key:

    animals[:color] => "black"
