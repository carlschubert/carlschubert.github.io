    #Cycle

    I have spent all week up to my neck in enumerables so In this weeks blog I am going to talk about one. It is called #cycle. In short it repeats the what ever it ois given in order until it receive some kind of break statement.

    cycle and Arrays

    When cycle is used on an array with a block, cycle sends each item in the array to the block. Once cycle gets to the end of the array cycle starts again at the beginning of the array unless there is a break somewhere in the block.

    fruit = %w{apple, pear, mango} fruit.cycle #=> apple, pear, mango, apple, pear, mango…. or fruit.cycle do |fruit| puts fruit break if fruit == "mango" end => apple pear mango => nil

    cycle and Hashes

    When cycle is used on a hash with a block, it works much like it does in an array. It sends each key/value pair to the block once cycle gets to the end of the array cycle starts again at the beginning of the array unless there is a break somewhere in the block.

    fruit = { :apple => ‘red’, :orange => ‘orange’ , :plum => ‘purple’} fruit.cycle {|fruit| p fruit} #=> [ :apple => ‘red’] [:orange => ‘orange’] [:plum => ‘purple’] ……….. or fruit.cycle do |type, color| puts "type: #{type} color: #{color}" break if type == :plum end => type: apple color: red type; orange color: orange type: plum color: purple => nil
