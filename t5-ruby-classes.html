    Ruby Classes

    This week I have been working a lot with classes so I thought I would write a short post about them.

    The first thing I usually consider when creating a new class first what are the built in attributes of the class. That is to say when I create a new object of class what does it do. The method initialize lets you set up what happens when a class is made.

    class Square
    def initialize(side)
    @side = side
    if @side < 0
    raise ArgumentError.new("Only positive numbers are allowed")
    end
    end
    end

    This class describes a square. When you create an object of this class you inculed the length of one side. The method initialize checks as the class is being created if the nember entered is positive. If it is not an error is raised because the square is not valid. It also defines the istance variable sides. Instance varibles are varibles availible to any method in the class.

    Say we want to know the area of our square.

    class Square
    def initialize(side)
    @side = side
    if @side < 0
    raise ArgumentError.new("Only positive numbers are allowed")
    end
    end
    def area
    (@side * @side)
    end
    end

    We can find the area of our square using

    square = Square.new(2)
    p square.area

    The method area was able to access side because we had labeled it an instance variable.
