    Margin Border and Padding

    Margin, borders and padding are an essential tool for defining how your web page elements will relate to each other using CSS. Above is what chrome dev tools displays to represent the three as they relate to the element.

    Margins

    The margin property is similar to the margin property common in word processors. Except rather than determining how far from the edge of a page content appears, margins in CSS determine how much transparent space appears after the border of an element. Margins are used to prevent website elements from crowding each other on the page.

    Padding

    Padding is similar to margin ,when I was first starting CSS I had trouble telling them apart. Instead of adding space outside of the border, the Padding property adds space between the border and the content of an element.

    Both margins and padding can be assigned different thickness margin:10px ,will apply a 10 pixel margin around the border for example. The color however can not be modified, they are always transparent.

    Border

    Once you understand margin and padding border is fairly straight forward. Borders appear between the margin and the padding and outline the element. They can be made thicker using the width value, altered a varity of ways to appear dashed dotted or double lined using the style value or have there color changed with the color.
